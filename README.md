# YouTube UI Clone

A responsive YouTube interface clone built with HTML and CSS.

## Features

- Clean, modern YouTube-like design
- Responsive layout for all devices
- Video grid layout
- Navigation header with search functionality

## Technologies Used

- HTML5
- CSS3
